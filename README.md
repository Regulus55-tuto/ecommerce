# Galaxy University Shop

대학생을 위한 삼성 갤럭시 제품 전문 판매 사이트



## 기술스택

React, javascript(es6), TypeScript, React-hook-form, Tailwind CSS, @headlessui/react, react-lazy-load-image-component, swiper
